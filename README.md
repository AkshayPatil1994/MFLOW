# MFLOW
This repository contains the code for the multi-fidelity Krigging for RANS and LES data as part of the Refmap project.

## Installation
To install the required dependencies, please run the following command:

```bash
cd mflow/
pip install -r requirements.txt
```

## Usage

Please consult the `mflow` directory for scripts and examples on how to use the code for multi-fidelity Krigging.
